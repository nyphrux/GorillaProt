<h1 align="center">Usage</h1>

### Downloading
- Go to the releases page [[Click]](https://github.com/nyphrux/GorillaProt/releases/latest)
- Download the latest release (*.dll)

### Installing
- Go to your game directory (where the game's .exe is located)
- Go to BepInEx/plugins (if it doesn't exist, download BepInEx)
- Place the downloaded .dll in the plugins folder

### Using - Ingame
- Everything is enabled by default, you dont have to do anything.
